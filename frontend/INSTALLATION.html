🚀 Installation Rapide

Étape 1 : Vérifier le Backend

Assurez-vous que votre API Spring Boot est démarrée sur :

http://localhost:8080

Étape 2 : Configurer l'URL de l'API

Ouvrez le fichier configuration/config.js et modifiez si nécessaire :

const API_BASE_URL = 'http://localhost:8080/api';

Étape 3 : Démarrer un Serveur Web

Option A - Python :

python3 -m http.server 8000

Option B - PHP :

php -S localhost:8000

Option C - Node.js :

npx http-server -p 8000

Étape 4 : Accéder à l'Application

Ouvrez votre navigateur et accédez à :

http://localhost:8000

⚠️ Configuration CORS

Assurez-vous que votre backend Spring Boot autorise les requêtes CORS depuis votre domaine frontend.

🐛 Dépannage

Problème : Erreur de connexion au serveur

Solution :

  • Vérifiez que le backend est démarré
  • Vérifiez l'URL dans configuration/config.js
  • Ouvrez la console du navigateur (F12) pour plus d'informations

Problème : Page blanche après connexion

Solution :

  • Vérifiez la console du navigateur (F12)
  • Vérifiez que tous les fichiers sont bien chargés
  • Vérifiez que les chemins des fichiers sont corrects
